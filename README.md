A set of tools and services to manage Waylearn's Bootcamps and in the future, DevRel.

This program is deployed and working on DevNet: [CozaAtw4wmRh1WT2zeLTu6jWYgrJotB7TRYdZZbusBNK](https://explorer.solana.com/address/CozaAtw4wmRh1WT2zeLTu6jWYgrJotB7TRYdZZbusBNK?cluster=devnet)
